1. Create a free account on Render.
2. Click New Web Service.
3. Connect your GitHub account and pick this repository.
4. Set the start line to: python backend/server.py
5. Set the region closest to you.
6. Add a name and click Create Web Service.
7. When the service is live, open the public address Render shows you and confirm the candidate page loads.
8. Open the same address with /admin at the end to reach the admin page.
9. Set the secret text for decoy generation by adding a value named LLM_API_KEY and pasting your key.
10. If you want a different admin login, set ADMIN_USER and ADMIN_PASSWORD before sharing the admin page.
