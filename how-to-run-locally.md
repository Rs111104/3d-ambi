1. Download and install Python from https://www.python.org/downloads/.
2. Download this project from https://github.com/Rs111104/3d-ambi and unzip it.
3. If you want decoy generation, copy your key from https://platform.openai.com/api-keys and put it after LLM_API_KEY in a file named .env by copying .env.example.
4. Double click run.bat on Windows or run.sh on Mac or Linux.
5. Open http://localhost:8080 for the candidate test and http://localhost:8080/admin for the admin dashboard.
6. Log in with username admin and password admin123!.
